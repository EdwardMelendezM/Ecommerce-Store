# Ecommerce Store

## Tools
- React JS
- Next JS 13
- Tailwind
- Typescript
- React-hot-toast
- Zustand

## Install
1. Clone this repository
2. Install all dependencies
```
  npm i
```
3. Change the variable enviaronment called NEXT_PUBLIC_API_URL like api
```
  NEXT_PUBLIC_API_URL=
```
4. Run the application
```
  npm run dev
```
